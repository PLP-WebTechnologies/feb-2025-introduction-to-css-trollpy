Introduction to CSS

integrating the cascading style sheet with HTML

What is CSS?

CSS (Cascading Style Sheets) is a styling language used to describe the presentation of HTML documents. It controls how web pages look in the browser, handling everything from colors and fonts to layouts and animations. CSS separates content (HTML) from presentation, allowing designers to create consistent styling across multiple pages. With CSS, you can control the visual appearance of text, images, and other elements without changing the underlying HTML structure.

How does CSS work?

  • CSS defines style rules for HTML elements
  • Each rule consists of a selector and a declaration block
  • The selector specifies which HTML elements the rule applies to
  • The declaration block contains the properties and values to style the selected elements
  • CSS rules are applied in the order they appear in the HTML document

CSS Examples

  • Changing the color of text: color: blue;
  • Setting a background color: background-color: lightgray;
  • Centering an element: text-align: center;
  • Creating a border: border: 1px solid black;
  • Adding a margin: margin: 10px;
  • Adding a padding: padding: 10px;

CSS Selectors

  • p: Selects all paragraph elements
  • h1, h2, h3, h4, h5, h6: Selects all heading elements
  • div: Selects all div elements
  • div p: Selects all paragraph elements within a div element
  • div > p: Selects only direct children of a div element that are paragraph elements
